HTTP status code 200

The given credentials were acceptable/valid (i.e. - the authentication succeeded). The body of the response will have the attributes of the authenticated User.

Media type: application/json

Type: object

Properties

    • prompt: required (one of yes, no)

      Flag to indicate whether to remind the user to add an MFA option
